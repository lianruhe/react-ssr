# react-ssr

React server side rendering

## Usage

```bash
# 安装依赖包
npm install

# 清除打包目录
npm run clean

# 打包
npm run compile

# 开启服务（监听模式）
npm run watch

# 开启服务（development）
npm run dev

# 开启服务（production）
npm run start
```

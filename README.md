# react-ssr

React server side rendering

## Usage

```bash
# install dependencies
$ npm install

# run server
$ npm start
```

## Development

### webapp development

```bash

$ npm run dev:web

```

### server development

```bash

# build webapp
$ npm run build

# server start
$ npm run dev

```
